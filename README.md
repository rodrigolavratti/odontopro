# 🚀 SaaS Profissional com Next.js 15

Sistema SaaS moderno desenvolvido com as principais tecnologias do ecossistema JavaScript e TypeScript, seguindo boas práticas de arquitetura, escalabilidade, segurança e experiência do usuário.

## 📋 Sobre o Projeto

Este projeto tem como objetivo a construção de um sistema SaaS completo, incluindo autenticação de usuários, gerenciamento de banco de dados, planos de assinatura, integração de pagamentos e painel administrativo.

A aplicação foi desenvolvida utilizando o ecossistema moderno do React e Next.js, explorando recursos avançados como Server Components, Server Actions e App Router.

---

### Front-end

### Back-end

- Next.js Server Actions
- API Routes

- Better Auth
- Login com Google OAuth

- Rotas Protegidas

### Pagamentos e Assinaturas

- Integração com Gateway de Pagamento
